# CS 188 - Introduction to Artificial Intelligence (Spring 2026)
### UC Berkeley | The Pac-Man Projects

![Pacman](https://inst.eecs.berkeley.edu/~cs188/sp26/assets/images/pacman_game.gif)

## Overview

The Pac-Man projects apply an array of AI techniques to playing Pac-Man. These projects teach foundational AI concepts such as informed state-space search, probabilistic inference, and reinforcement learning — concepts that underlie real-world applications in natural language processing, computer vision, and robotics.

**Course page:** https://inst.eecs.berkeley.edu/~cs188/sp26/projects/

---

## Projects

| Project | Topic | Description |
|---------|-------|-------------|
| Project 0 | Python, Setup & Autograder | Introduction to Python and the autograder system |
| Project 1 | Search | Depth-first, breadth-first, uniform cost, and A* search algorithms for navigation problems |
| Project 2 | Multiagent Search | Minimax and expectimax algorithms, evaluation functions |
| Project 3 | Reinforcement Learning | Value Function, Q-learning, and Approximate Q-learning |
| Project 4 | Bayes Nets & HMMs | Probabilistic inference, forward algorithm, and particle sampling |
| Project 5 | Machine Learning | Perceptron, neural networks, and recurrent models for classification |
